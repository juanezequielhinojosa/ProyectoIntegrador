Configuración de una red LAN en entorno simulado

¿Cómo lo vas a hacer?

Usando software de simulación de redes (como Cisco Packet Tracer o simuladores similares), los estudiantes configurarán una red LAN básica. Primero, el docente brindará una guía paso a paso para configurar dispositivos (routers, switches, computadoras virtuales). Luego, cada grupo creará una red con direcciones IP asignadas, establecerá un servidor de archivos básico y realizará pruebas de comunicación entre nodos. Se fomentará que identifiquen problemas comunes y apliquen soluciones básicas, mientras se registran evidencias (capturas de pantalla, informes)
