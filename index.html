¿De qué se trata?

Invitamos a docentes de los espacios curriculares a participar de un proyecto colaborativo donde exploraremos la arquitectura, conectividad y seguridad de una computadora construida conceptualmente desde cero. Este proyecto busca integrar saberes desde diferentes asignaturas para construir conocimientos significativos.
